# What is Linear Search?
- **Linear Search** is a basic searching technique used to find a target element by checking each element in a list one by one.
- The search begins from the **first element** and continues sequentially until the target is found or the list ends.
- Linear Search does **not require the list to be sorted**, making it simple but inefficient for large datasets.
- In the worst case, all elements are examined, resulting in **O(n)** time complexity.

## Key Characteristics of Linear Search

- **Sequential Access**  
  Elements are examined one after another.

- **Works on Unsorted Data**  
  No prior sorting is required.

- **Simple and Intuitive**  
  Easy to understand and suitable for beginners.

- **Early Termination**  
  Stops immediately when the target element is found.

- **Time Complexity**  
  - Best Case: **O(1)**  
  - Worst Case: **O(n)**
