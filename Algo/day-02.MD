## Question:
### Searching Techniques – Linear Search
### PROBLEM:
     Given a list of elements and a target value, determine whether the target exists in the list and return its position.

# Algorithm:
## Step 1: Start
     - Begin the search from the first element of the list.
## Step 2: Compare
     - Compare the current element with the target value.
## Step 3: Condition Check
     - If the current element is equal to the target:
     - Return the current index (position found).
     - Stop the algorithm.
## Step 4: Move          
     - Move to the next element in the list.
## Step 5: Repeat
     - Repeat Step 2 to Step 4 until:
     - The target element is found, or
     - The last element of the list is reached.
## Step 6: End
     - If the end of the list is reached and the target is not found:
     - Declare that the target value does not exist in the list.
